üöÅ UAV Intelligent Inspection Key Technologies and 3D Platform

Power Grid Inspection Project | Student Lead
Overview: This project uses airborne LiDAR to complete high-precision holographic digital modeling of power lines and towers in Jiangsu Province. Based on 3D point cloud data, it achieves real-time centimeter-level trajectory planning for multi-rotor UAV power tower inspection. Finally, a complete power tower inspection system was developed to achieve digital, automated, and refined inspection.
Responsibilities: (1) Flight space grid modeling based on 3D point cloud data and UAV hovering accuracy via ArcGIS interface; (2) Constructing objective functions for minimum energy consumption and minimum lighting cost, considering natural wind and lighting effects; (3) Improving Ant Colony + A* hybrid algorithm for real-time path planning under different times/wind directions; (4) Developing power inspection system prototype tools including path planning, safety analysis, knowledge base, parameter library, and visualization engine.

Overview: This project focuses on the Tianyi integrated development environment, researching code debugging and runtime monitoring technology for multi-task parallel programs based on the Tianyi embedded operating system.
